# 艾薇拉 II - 恐怖劇場　手冊要點索引

> 整理自當年**軟體世界（Softworld）中文說明書**（骨灰集散地掃描保存）與遊戲實機。
> 這份是「上手速查」；完整世界觀見 [README](../README.md)、逆向與系統細節見 [elvira2_research.md](elvira2_research.md)。

## 一、開場流程

1. 進遊戲會先播**片頭 credits**（Horror Soft／Accolade／製作團隊），按 `Enter` 可跳過。
2. 接著出現 **「Restore a saved game? Yes / No」**（讀取存檔嗎？是／否）：
   - 選 **Yes** → 載入舊進度。
   - 選 **No** → 重新開始，進入角色選擇。
3. **選擇角色類型（Select Character Type）**，四種職業各有不同起始屬性：

| 英文 | 官方手冊譯名 | 本中文化 | 特色 |
|---|---|---|---|
| Stuntman | 替身演員 | 特技替身 | 力量(STR 10)、武器技能(WS 10)高，戰士型 |
| Private Eye | 私家偵探 | 私家偵探 | 屬性平均，泛用 |
| Computer Programmer | 程式設計師 | 電腦程式設計師 | 智慧(INT 10)、抗魔力(MR 10)高，法師型 |
| Knife Thrower | 特技人員 | 飛刀手 | 命中率(ACC 10)、體質(CON 10)高，敏捷型 |

## 二、八大屬性（升級時成長，起始值固定）

| 縮寫 | 英文 | 中文 | 說明 |
|---|---|---|---|
| WS | Weapons Skill | 武器技能 | 持武器時，讓你使用武器更得心應手 |
| ACC | Accuracy | 命中率 | 攻擊敵人時武器的命中率 |
| WP | Will Power | 意志力 | 意志力堅強者，往往是贏家 |
| MR | Magic Resist | 抗魔力 | 愈高，抵抗魔法的能力愈強 |
| PR | Poison Resist | 抗毒性 | 抵抗毒氣或各種有毒物品的能力 |
| CON | Constitution | 體質 | 健康程度，良好體質可增加與維持生命力 |
| STR | Strength | 力量 | 愈大，對敵人的傷害力愈大 |
| INT | Intelligence | 智慧 | 決定技巧的熟練與否 |

> 遊戲底部狀態列顯示的是：**LEV**（等級，上限 10）、**P.P**（心靈力／PSY）、**H.P**（生命值）、**EXP**（經驗值，上限 9999）。

## 三、法術調配系統（spell mixing）

Elvira 系列的招牌：法術不是學會就能放，而是要**把材料放到「法術頁（spell page）」上調配**。材料本身是謎題——你得從物品的「材料類型描述」推敲該放什麼。常見法術：

- **治療類**：治療之手、草藥療癒、治傷術
- **攻擊類**：火球術、閃電、冰鏢術、聖光爆、新星術
- **輔助類**：祝福、勇氣術、保護術、附魔武器、念力、水下呼吸術
- **高階**：復活術（再生／召喚／結合三階段）、束縛惡魔、吸收魔法、破魔術

## 四、操作與存讀檔

- **移動**：右下角方位羅盤（東南西北 + UP/DOWN）。
- **物品**：底部道具欄；`INV` 開背包。
- **互動**：滑鼠點場景物件／道具，執行查看、拿取、使用等動作。
- **存檔**：在可操作場景才能存（開場 cutscene 期間存檔被擋）。ScummVM 下 `Ctrl+F5` 開選單存讀。

## 五、現代玩家友善化熱鍵（本中文化新增）

| 鍵 | 功能 |
|---|---|
| `TAB` | 動態地圖（白格＝你、黃點＝寶物、紅點＝怪物）|
| `F8` | 除霧全圖 |
| `F7` | 無敵模式（HP／心靈力保持滿）|
| `F5` | 立即補血 |
| `F6` | 把當前房間可拿物品塞進背包 |

## 六、三座攝影棚（關卡）

穿過三座「戲中戲」活過來的攝影棚，湊齊道具與法術，救出艾薇拉、擊退地獄犬：

1. **一號棚**：昆蟲洞窟（巨蟻、巨蛛、巨蠍…）
2. **二號棚**：鬼屋（服裝間女巫、殭屍、吸血鬼…）
3. **三號棚**：地牢（骷髏、報喪女妖、死靈法師…）
